「ベランダに蜂の巣みたいなものがある」「庭に蜂がたくさん飛んでいる」──そんなとき、まず知るべきなのは**巣の種類**です。蜂の巣は種類によって危険度がまったく異なり、対処法も変わります。

この記事では、福岡で蜂駆除を専門に行う「蜂駆除あらい」が、**スズメバチ・アシナガバチ・ミツバチ**の巣の見分け方を写真付きで詳しく解説します。

> **注意**: 蜂の巣を発見しても、絶対に2メートル以内に近づかないでください。特にスズメバチは巣の近くを通るだけで攻撃してくることがあります。

#### スズメバチの巣の特徴 ── 最も危険、見つけたら即プロへ

**形状・模様**

スズメバチの巣は、完成形では**丸いボール状（球体）**をしています。表面には灰色〜茶色の**マーブル模様（波状の縞模様）**が特徴的で、これは木の皮や樹皮を噛み砕いて作るためです。

#### 蜂に刺されたら、まずやるべき5つのステップ

蜂に刺されたら、慌てずに以下の手順で対処してください。

**Step 1. その場を静かに離れる**

まず、蜂がいる場所から**静かに、低い姿勢で**数十メートル離れてください。走ったり、手で振り払ったりすると、蜂を刺激してさらに攻撃される危険があります。

スズメバチに刺された場合は特に注意が必要です。スズメバチは刺すときに**警報フェロモン**を出し、仲間を呼び寄せます。すぐにその場を離れないと、複数の蜂に襲われる可能性があります。

**Step 2. 針が残っていれば取り除く**

ミツバチに刺された場合、**針が皮膚に刺さったまま残る**ことがあります（スズメバチ・アシナガバチの針は残りません）。

針が残っている場合は、**指でつまんで抜こうとしないでください**。針の根元には毒嚢（どくのう）がついており、つまむと毒嚢を圧迫してさらに毒が注入されてしまいます。

正しい方法は、**クレジットカードや爪など硬くて薄いもので、横方向にこそぐようにして弾き飛ばす**ことです。

**Step 3. 毒を絞り出す**

刺された箇所の周囲を指でつまみ、**毒を血液と一緒に絞り出し**ます。ポイズンリムーバー（毒吸い出し器）があればより効果的です。アウトドアに出かけることが多い方は、ドラッグストアやネット通販で1,000円前後で購入できるので、1つ持っておくことをおすすめします。

**重要: 口で吸い出すのは絶対にNG**です。蜂の毒は水溶性のため唾液に溶けやすく、口の中の粘膜や歯茎の傷から毒が体内に入り込む危険があります。口内がしびれたり腫れたりする事例が報告されています。

**Step 4. 流水で洗い、冷やす**

刺された箇所を**流水でしっかり洗い流し**ます。蜂の毒は水溶性なので、流水で洗うことで毒の一部を洗い流す効果があります。

その後、**保冷剤や氷のう、なければ冷水で濡らしたタオル**で患部を冷やしてください。冷やすことで血管が収縮し、毒が体内に広がるスピードを遅くする効果があります。

**Step 5. 薬を塗って30分〜1時間安静にする**

患部に**抗ヒスタミン成分を含むステロイド軟膏**を塗ります。抗ヒスタミン成分がかゆみを抑え、ステロイド成分が炎症とアレルギー反応を抑えます。ドラッグストアで「虫刺され用」として販売されているステロイド軟膏（ムヒアルファEXなど）が適しています。

薬を塗ったら、**30分〜1時間は安静にして体調の変化を観察**してください。この間にアナフィラキシーの症状が出ることがあるため、一人きりにならないようにしましょう。

#### こんな症状が出たら、すぐに病院へ（救急車を呼ぶべきケース）

蜂に刺された後、以下の症状が出た場合は**すぐに救急車（119番）を呼んでください**。

**アナフィラキシーショックとは**

アナフィラキシーショックとは、蜂の毒に対する**重篤な全身性アレルギー反応**です。蜂に刺されたことで体内に作られた抗体が、2回目以降の刺傷時に過剰反応を起こし、血圧低下・呼吸困難・意識障害などを引き起こします。

アナフィラキシーは刺されてから**数分〜30分以内**に急速に進行し、適切な処置がなければ死に至ることがあります。日本では毎年約20名が蜂刺傷によるアナフィラキシーショックで亡くなっています。

**危険な症状チェックリスト**

以下の症状が1つでも現れたら、**迷わず救急車を呼んでください。**

- 刺された場所以外にも蕁麻疹（じんましん）が広がる
- 顔や唇、まぶたが腫れる
- 息苦しい、喉が締め付けられる感じがする
- 声がかすれる
- めまい、ふらつき、立っていられない
- 吐き気、嘔吐、腹痛、下痢
- 動悸が激しい
- 冷や汗が出る
- 意識がもうろうとする

**エピペン（アドレナリン自己注射薬）について**

過去に蜂に刺されてアレルギー症状が出たことがある方は、医師に相談して**エピペン（アドレナリン自己注射薬）**の処方を受けておくことをおすすめします。

エピペンはアナフィラキシー発症時に太ももに注射することで、症状の進行を一時的に抑える薬です。ただし、エピペンの効果は一時的（15〜30分程度）なので、**使用後は必ず救急車を呼び、医療機関を受診してください。**

また、アナフィラキシーには**遅発性反応（二相性反応）**が起こることがあります。最初の反応が治まった後、8〜12時間後に再び症状が現れるケースがあるため、蜂に刺されて一度でも症状が出た場合は、当日中は医師の観察下にいることが推奨されます。

**巣の大きさ・場所による料金の変動**

蜂駆除の料金は、巣の大きさと設置場所によって変動します。

| 条件 | 料金への影響 | 理由 |
|------|------------|------|
| 巣が小さい（初期・直径10cm以下） | 安い | 蜂の数が少なく、作業が短時間で済む |
| 巣が大きい（直径30cm以上） | 高い | 蜂の数が多く、危険度・作業時間が増す |
| 手の届く場所（軒下・ベランダ等） | 標準 | 特別な機材が不要 |
| 高所（2階以上の屋根裏、軒下） | +5,500〜11,000円 | 梯子や高所作業車が必要 |
| 屋根裏・天井裏の中 | +11,000円〜 | 建物内部への進入作業が必要 |
| 地中（オオスズメバチに多い） | +11,000円〜 | 掘削作業が必要 |
| 壁の中 | +11,000〜22,000円 | 壁の開口・復旧が必要な場合がある |

**季節による料金の変動**

蜂の巣は時期によって大きさが異なるため、駆除の難易度と料金も季節によって変わります。

| 時期 | 巣の状態 | 料金傾向 |
|------|---------|---------|
| 4〜5月（春） | 女王蜂が1匹で巣作り中。巣は小さい | **最も安い** |
| 6月（初夏） | 働き蜂が増え始める。巣も成長中 | やや上がる |
| 7〜9月（真夏〜初秋） | 巣が最大化。蜂の数・攻撃性ともにピーク | **最も高い** |
| 10月（秋） | 活動は続くが、ピークは過ぎる | やや下がる |
| 11〜3月（冬） | 蜂は活動停止。巣は空 | 空き巣の撤去のみなら安い |

**つまり、蜂駆除は「早ければ早いほど安い」**のです。巣が小さいうちに駆除すれば、費用も危険も最小限で済みます。

- 料金体系が不透明な業者ほど、トラブルのリスクが高い

#### 「安すぎる業者」に注意！ 悪質業者の見分け方

**「3,000円〜」は本当か？**

インターネット上には「蜂駆除3,000円〜」「最安値挑戦中！」といった広告が目立ちます。しかし、**実際に3,000円で済むことはほぼありません**。

蜂駆除には、防護服・殺虫剤・高所作業の機材・出張費など、最低限のコストがかかります。プロが安全に作業を行うには、数千円では採算が合いません。

「3,000円〜」の「〜」は「その金額から始まる」という意味であり、実際にはそこから何倍もの費用がかかることが常態化しています。

**悪質業者の典型的な手口**

NHKでも報道されたように、悪質な駆除業者（蜂駆除に限らず、水道修理なども含む）の典型的な手口は以下のとおりです。

1. **格安広告で集客**: 「蜂駆除2,900円〜」「業界最安値」でリスティング広告を出す
2. **現地で高額請求**: 巣を見せて「これは特殊な作業が必要」「追加費用がかかる」と言い、当初の数倍〜数十倍の料金を請求
3. **断りにくい状況を作る**: 「もう作業を始めた」「キャンセル料がかかる」と言って支払いを迫る
4. **作業が不十分**: 巣を完全に除去しないため、再発する

**信頼できる業者の選び方チェックリスト**

- [ ] 料金表が事前に公開されている
- [ ] 蜂の種類・巣の大きさ・場所ごとの料金目安がわかる
- [ ] 見積もりが無料で、見積もり後のキャンセルが可能
- [ ] 追加料金の条件が明記されている

- [ ] 口コミ・施工事例が確認できる
- [ ] 電話対応が丁寧（料金を聞いてもはぐらかさない）

#### 福岡市は行政で蜂駆除をしてくれる？

福岡市では、**行政による蜂の巣駆除は行っていません**。蜂の巣ができた場合は、市民の方が自費で業者に依頼する必要があります。

各区役所の生活環境課に相談すると、蜂の種類の判別や注意事項についてアドバイスを受けることはできますが、駆除作業そのものは民間の業者に依頼することになります。

なお、福岡県内の一部自治体（飯塚市など）では、スズメバチの巣の駆除費用に対する**補助金制度**が設けられている場合があります。お住まいの市町村にご確認ください。

  │     ├── アシナガバチ事例 → 蜂の知識（アシナガバチ関連）+ 地域ページ
  │     └── ミツバチ事例 → 蜂の知識（ミツバチ関連）+ 地域ページ
  │
  └── 地域ページ
        ├── 福岡市の蜂駆除 → その地域の施工事例 + 蜂の知識

        └── ...
```

#### リンクテキストのベストプラクティス

| NG | OK | 理由 |

### 3-5. SEO設定チェックリスト

```
■ タイトル・メタ情報

  □ H2・H3にも関連キーワードが含まれている

■ 本文
  □ 文字数が1,000文字以上（施工事例は800文字以上）
  □ ターゲットキーワードが本文中に自然に含まれている
  □ 画像にalt属性が設定されている

■ 内部リンク

  □ リンクテキストにキーワードが含まれている

  □ エリアタグが設定されている（施工事例の場合）

  □ スマホでの表示を確認（レスポンシブ対応）
  □ 画像が適切に表示されている
  □ リンクが正しく動作する
  □ 誤字脱字がない

### 4-2. 施工実績記事のリライト方針

現状の施工実績記事は以下の問題を抱えている。

| 問題点 | 現状 | 改善案 |
|--------|------|--------|
| 文字数が少ない | 300〜500文字程度 | **800〜1,200文字**に拡充 |
| 地域名がない | 記載なし | **市区町村名**を明記（可能なら町名まで） |
| 蜂の種類の説明がない | 種類名のみ | その蜂の**特徴・危険度の簡単な解説**を追加 |
| 料金の記載がない | 記載なし | **実際の料金（税込）**を明記 |
| 所要時間がない | 記載なし | **作業時間**を明記 |
| 巣の詳細がない | 記載なし | **巣の大きさ・場所**を明記 |
| ビフォーアフター写真がない | 写真が少ない or なし | **駆除前・駆除後の写真**を追加 |
| 内部リンクがない | リンクなし | **関連記事・料金ページ・CTA**を追加 |
| カテゴリが「ブログ」 | 「ブログ」のみ | **「施工事例」に変更 + 適切なタグ設定** |

#### 施工実績記事に追加すべき情報（具体例）

既存記事「駆除実績 8/5 スズメバチ」（記事ID: 209）の場合:

```
【現状の内容】
・スズメバチの巣を駆除した（写真1〜2枚）
・文字数: 約300〜500文字

【リライト後のイメージ】
・タイトル: 【施工事例】福岡市○○区でスズメバチの巣を駆除｜軒下に30cmの巣
・施工概要テーブル（日付・エリア・種類・場所・大きさ・時間・料金）
・ビフォーアフター写真（2〜4枚）
・お客様からのご依頼内容（100字）
・現地の状況と駆除の詳細（200〜300字）
・お客様へのアドバイス（100字）

```

**注意**: 過去の施工実績について、地域名・料金・所要時間が不明な場合は、クライアント（あらい代表）に確認の上で記入する。記憶にない場合は無理に書かず、わかる範囲で充実させる。
