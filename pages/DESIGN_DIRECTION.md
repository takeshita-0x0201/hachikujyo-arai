# デザインディレクション（全ページ共通）

**方針**: 「信頼できる町の蜂駆除屋さんのホームページ」  
**トーン**: 清潔感のあるシンプル。行政サイト・病院サイト寄り。  
**禁止**: マーケティング会社が作った「集客サイト」感  
**作成日**: 2026-04-02

---

## 1. デザイン原則

### やること / やらないこと

| 要素 | ◎ やること | ✗ やらないこと |
|------|-----------|--------------|
| **全体の印象** | 誠実・清潔・素朴・安心 | 派手・売り込み・キラキラ・圧迫感 |
| **背景** | 白ベース + 薄いグレー（#F5F5F5）のセクション交互 | グラデーション、柄背景、動画背景 |
| **色数** | 3色以内（紺 + 白 + 赤/橙のCTAのみ） | 多色使い、ネオンカラー、ゴールド |
| **フォント** | Noto Sans JP、16px以上、行間1.8 | 装飾フォント、筆記体、極小フォント |
| **レイアウト** | 1カラム中心、余白たっぷり | 3カラム、情報詰め込み、ギチギチ |
| **画像** | 実際の施工写真、代表の写真（加工なし） | ストック写真、イラスト、過度な加工 |
| **アニメーション** | なし。あってもフェードイン程度 | パララックス、スライドイン、カウントアップ |
| **CTA** | 大きく・はっきり・静的 | 点滅、ポップアップ、追従バナー過多 |
| **装飾** | 最小限の罫線、背景色の切替のみ | 影(box-shadow)、角丸カード大量、アイコン過多 |
| **料金表** | HTMLテーブル、全額掲載、シンプル | 画像化した料金表、「要相談」で隠す |
| **口コミ** | 静的なリスト表示 | カルーセル、スライダー、星アニメーション |
| **施工事例** | 静的なグリッド or リスト | スライダー、ライトボックス |
| **文章** | 事実を淡々と。短文。「です・ます」調 | 煽り文、感嘆符過多、誇大表現 |

## 6. コピーライティングのトーン調整

### Before（現在の制作案のトーン）

```
❌ 「蜂の巣を見つけて不安なあなたへ。今日中に解決します！」
❌ 「圧倒的な実績と信頼！」
❌ 「今すぐお電話を！」
❌ 「お客様満足度98%！」
```

### After（調整後のトーン）

```
◎ 「蜂の巣の駆除は、蜂駆除あらいにご相談ください。」
◎ 「料金は全て事前にお伝えします。」
◎ 「お電話ください。 090-8451-0837」
◎ 「福岡県を中心に、年間○○件の駆除を行っています。」
```

### トーンのルール

| ルール | 説明 |
|--------|------|
| **感嘆符は最小限** | 1ページに2〜3個まで。「！！」の連続は禁止 |
| **煽らない** | 「不安なあなたへ」「今すぐ」「圧倒的」は使わない |
| **事実を述べる** | 「24時間対応しています」「料金は○○円です」と淡々と |
| **です・ます調** | 敬語だが堅すぎない。親しみやすい丁寧語 |
| **主語は「当店」** | 「私たち」「弊社」ではなく「当店」。町のお店感 |
| **数字で語る** | 「多くの実績」→「年間○○件」。具体的に |
