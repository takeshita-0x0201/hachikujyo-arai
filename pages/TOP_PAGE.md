## 6. 実装時のチェックリスト

### SEO

- [ ] title タグが正しく出力されている（echo バグの修正）
- [ ] meta description が120〜160文字で設定されている
- [ ] H1 タグがページに1つだけ存在する（テキストで）
- [ ] canonical URL が正しく設定されている
- [ ] OGP タグが正しく設定されている
- [ ] 構造化データ（LocalBusiness / FAQPage / Service）がエラーなく実装されている
- [ ] 画像に適切な alt 属性が設定されている
- [ ] ページ内リンク（アンカー）が正しく動作する

### パフォーマンス

- [ ] 画像がWebP形式 or 適切に圧縮されている
- [ ] FV画像にプリロードが設定されている（`<link rel="preload">`）
- [ ] 不要なCSS/JSが読み込まれていない
- [ ] Largest Contentful Paint（LCP）が2.5秒以内

### コンバージョン

- [ ] 電話番号が全箇所で統一されている（`090-8451-0837`）
- [ ] 全ての電話番号がtap-to-callに対応している
- [ ] CTA ボタンが8箇所以上配置されている
- [ ] モバイル固定CTAバーがSPで正しく表示されている
- [ ] モバイル固定CTAバーがPCで非表示になっている
- [ ] CTAボタンの色が他の要素と十分にコントラストがある
- [ ] お問い合わせフォームへのリンクが正しい（`/contact`）

### レスポンシブ

- [ ] PC（1024px以上）で正しく表示される
- [ ] タブレット（769〜1024px）で崩れない
- [ ] SP（768px以下）で正しく表示される
- [ ] SP（375px幅 / iPhone SE）で崩れない
- [ ] ヘッダーの固定表示がスクロール時に正しく動作する

### アクセシビリティ

- [ ] 全ての画像に alt 属性がある
- [ ] CTAボタンに十分なタッチターゲットサイズがある（44px以上）
- [ ] フォントサイズが小さすぎない（本文16px以上）
- [ ] 色のコントラスト比が WCAG AA 基準を満たす
