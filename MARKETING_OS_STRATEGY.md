## Step 1: WHY — 目的・目標

```
━━━ WHY: 目的・目標 ━━━
■ ビジネス目的: Web経由の問い合わせ件数を大幅に増加させ、赤字脱却 → 事業を黒字化する
■ 定量目標: 蜂シーズン（5〜10月）に月間40〜50件の問い合わせを獲得
  （岡山で実現していた水準を福岡でも再現する）
■ 期限: 2026年の蜂シーズン（5〜10月）に間に合わせる → 残り1〜2ヶ月
■ Why Now: 
  - 現在赤字で事業継続の危機
  - 蜂のシーズン（5〜10月）開始まで時間がない
  - 岡山での実績（月40〜50件）があるので、福岡でも再現可能なはず
  - 福岡は岡山より人口が多く、潜在需要も大きいはず
```

### WHYから導かれる制約条件

| 条件 | 内容 |
|------|------|
| 時間制約 | 2026年5月のシーズン開始まで約1ヶ月。即効性のある施策が最優先 |
| 予算制約 | 赤字状態のため、大規模な広告投資は困難。低コスト施策中心 |
| リソース制約 | 個人事業（推定）。サイト改修は外注 or 自力 |
| 成功基準 | 月40〜50件の問い合わせ = 1日1.3〜1.7件。検索経由+電話が主導線 |

### 逆算思考

月40件の問い合わせを得るために必要な数字（業界平均CVRから推計）:

```
月40件の問い合わせ
  ÷ CVR 10%（電話タップ率。業界成功事例の目標値）
  = 月400セッション必要
  ÷ CTR 5%（検索結果からのクリック率）
  = 月8,000回の検索結果表示が必要
```

→ 「福岡 蜂駆除」関連キーワード群で月8,000インプレッションは十分達成可能な規模。
→ **問題はそもそも検索結果に表示されていないこと**。ここが最大のボトルネック。

### 3-3. Sufficient Demand, Right to Win 評価

**需要A（緊急駆除）の評価**:

| 評価軸 | 判定 | 理由 |
|--------|------|------|
| **Sufficient Demand** | ◎ | 福岡県の人口は岡山県の約2.5倍。「福岡 蜂駆除」の検索ボリュームは十分。月40〜50件は達成可能な規模 |
| **Money** | ◎ | 蜂駆除は緊急性が高く、価格感度が相対的に低い。平均単価3〜4万円で十分な粗利が見込める |
| **Right to Win** | △→○ | 現状は競合に大きく負けているが、**サイト改善で逆転可能**。①岡山での経験・ノウハウ、②24時間対応、③料金表の透明性、④個人事業ならではの人柄、は既にある。足りないのは「それをWebで伝える力」だけ |

**結論**: 需要も金も十分。勝てる要素もある。**問題は「伝え方」と「見つけてもらう仕組み」が壊滅的に弱いこと**。

## Step 4: WHAT — 価値設計

### 4-1. コンセプト設計

競合との差別化を踏まえ、蜂駆除あらいのコンセプトを設計する。

**大手チェーンとの差別化軸**:
- 大手 = 全国対応・実績数・フリーダイヤル・組織力
- あらい = **地域密着・代表が直接対応・明朗会計・人のぬくもり**

```
━━━ コンセプト ━━━

■ 一言コンセプト:
  「福岡の蜂トラブル、代表が直接駆けつけて即解決。料金は全部事前にお伝えします。」

■ コンセプトの3要素:
  需要: 蜂の巣を見つけて怖い・困っている福岡県民
  ベネフィット: 今日中に恐怖から解放される + ぼったくられない安心
  独自性: 代表自らが対応する地域密着の安心感 + 料金完全公開
```

**なぜこのコンセプトか**:
- 大手チェーンの弱点は「誰が来るかわからない」「結局いくらかわからない」
- 個人事業の最大の武器は「人」。代表が直接来る安心感は大手にはない
- 岡山で月40〜50件の実績 = 経験とノウハウは十分にある

### 4-2. ベネフィット設計

```
━━━ ベネフィット（3層構造）━━━

■ 特徴（Feature）— 客観的な仕様:
  1. 24時間365日対応
  2. 福岡に拠点
  3. 季節別・種類別の料金表を事前公開
  4. 代表が直接対応（下請けなし）
  5. 蜂駆除 + 鍵交換のワンストップ
  6. 現金 + クレジットカード対応

■ メリット（Merit）— 特徴がもたらす直接的利点:
  1. 連絡すればすぐ来てくれる（待たされない）
  2. 福岡市内なら最短で到着（近い）
  3. 電話で概算料金がわかる（不安がない）
  4. 経験豊富なプロが来る（安心）
  5. 住まいのトラブル全般を任せられる
  6. 手持ちがなくてもOK

■ ベネフィット（Benefit）— 得られる感情価値:
  1. 「蜂の恐怖から今日中に解放される安堵感」
  2. 「地元のプロが来てくれるという安心感」
  3. 「いくらかかるか事前にわかる、ぼったくられない安心感」
  4. 「ベテランの人が来て確実にやってくれる信頼感」
  5. 「困ったときはここに電話すればいいという心の拠り所」

■ ユニークベネフィット（競合にはないもの）:
  → 「大手チェーンの"知らない誰か"ではなく、
     地元・福岡のベテラン代表が、直接あなたの家に来てくれる。
     料金も全部事前に伝えてくれる。安心できる"顔の見える駆除屋さん"。」
```

### 4-3. RTB設計

```
━━━ RTB（信じられる根拠）━━━

■ ロジック（論理的根拠）:
  1. 季節別・種類別の料金表を全公開 → 「明朗会計」の証拠
  2. 24時間対応の実態（直接電話が繋がる） → 「すぐ来る」の証拠
  3. 代表のプロフィール・経歴の開示 → 「経験豊富」の証拠
  4. 蜂の種類別の駆除方法を解説 → 「専門知識がある」の証拠

■ 実績（社会的証明）:
  ※現在ほぼゼロ → 以下を早急に構築する
  1. 施工事例（写真付き）を蓄積 → 月4件以上
  2. お客様の声を収集 → 施工後に依頼
  3. Googleビジネスプロフィールの口コミ → 獲得の仕組み化
  4. 「年間○○件の施工実績」を数値化して掲示
  5. 岡山での実績を「前職・前拠点での経験」として言及
```

### 4-4. オファー設計

```
━━━ オファー設計 ━━━

■ フロントエンド（入口オファー）:
  「電話で無料相談 + 概算見積もり」
  → ハードル: ゼロ（無料、電話1本）
  → 価値: 蜂の種類や状況を伝えれば概算金額がわかる
  → 目的: まず電話してもらう → 現地調査 → 成約

■ バックエンド（本命サービス）:
  蜂駆除施工（平均3〜4万円）

■ ハードル低減策:
  1. 「電話で概算見積もりOK」→ 来てもらう前に金額感がわかる
  2. 「見積もり後のキャンセルOK」→ 断れるという安心感
  3. 「クレジットカード対応」→ 手持ちがなくてもOK
  4. 「再発時の対応あり」→ 駆除後の不安を解消
  （※「再発保証」を正式に導入できると競合優位になる）

■ 限定感の演出:
  → 蜂駆除は本質的に「今すぐ」の需要。限定感は不要。
  → むしろ「今日中に対応できます」の即時性が最大の訴求。

■ 価値 > ハードル の設計:
  価値: 蜂の恐怖から解放 + 家族の安全確保 = プライスレス
  ハードル: 電話1本 + 3〜4万円 + 作業時間30分〜1時間
  → 圧倒的に価値 > ハードル。問題は「価値が伝わっていないこと」。
```

## Step 5: HOW — コミュニケーション戦略

### 5-1. MOL / MOA 設計

```
━━━ MOL → MOA 設計 ━━━

■ MOL（広告接触前の内界 — 検索結果を見る瞬間）:

  「ベランダ（or 軒下 / 庭の木）に蜂の巣を見つけた。
   子供がいるし怖い。自分で取るのは危険そう。
   スマホで「福岡 蜂駆除」って検索した。
   いくつか出てきたけど、どこがいいかわからない。
   とにかく早く来てくれて、ぼったくられないところがいい。
   3,000円〜って書いてあるサイトがあるけど、
   実際はもっとかかるんでしょ？不安...」

  → 内界の状態: 恐怖 + 焦り + 不安（料金・業者選び）
  → 求めているもの: 即時性 + 料金の透明性 + 信頼感

■ MOA（「よし、あらいに電話しよう！」の瞬間の内界）:
```

### 5-2. コミュニケーションストーリー（7ステップ）

### 5-3. ビッグアイデア発想

ターゲットの内界に憑依し、「つい見てしまう」「続きを見てしまう」アイデアを発想。

```
━━━ ビッグアイデア案 ━━━

■ 案1:「料金、全部見せます。」
  コンセプト: 蜂駆除業界の「結局いくらかわからない問題」を真正面から解決
  展開: 料金表を検索結果のdescriptionにも入れる

  なぜ刺さるか: 3,000円〜と書いて実際は高額、という業界不信への回答
  MOAへの繋がり: 料金が明確 → 安心 → 電話

■ 案2:「福岡の蜂駆除屋さん、あらいです。」
  コンセプト: 大手チェーンではなく「地元のおっちゃん」感で差別化
  展開: 代表の顔写真 + 名前 + 「私が直接伺います」
       地域名を前面に出したローカル感
  なぜ刺さるか: 知らない大手より、顔が見える地元の人の方が安心
  MOAへの繋がり: 人柄がわかる → 信頼 → 電話

■ 案3:「蜂の巣、見つけたら電話1本。今日中に解決。」
  コンセプト: 究極のシンプルさ。悩む必要なし、電話すればOK
  展開: ファーストビューに電話番号ドーン
       「見つけた→電話→今日中に駆除完了」の3ステップ
  なぜ刺さるか: 緊急時の人は情報過多に耐えられない。シンプルが最強
  MOAへの繋がり: 簡単さ → 行動障壁が消える → 電話
```

**推奨**: **案1と案2のハイブリッド**

「福岡市の蜂駆除屋あらいです。料金は全部お伝えします。」
→ 地域密着の安心感 + 料金透明性の安心感 = ダブルの安心で差別化
