## はじめに

本レポートでは、現在のWebサイト（hachikujyo-arai.com）の課題を整理し、リニューアル後のデモサイトでどのように改善されるかをご説明します。

## 1. 現在のサイトの課題

### 1-1. 検索で見つからない

| 問題 | 詳細 |
|------|------|
| **Googleに3ページしか認識されていない** | サイト内に39ページあるにもかかわらず、Googleが認識しているのは3ページのみです。titleタグの出力バグが原因と考えられます |
| **「福岡 蜂駆除」で検索しても表示されない** | 検索上位は全国チェーンやポータルサイトが占めており、蜂駆除あらいは圏外です |
| **Google Analytics等の計測ツールが未導入** | サイトへの訪問者数、電話タップ数、問い合わせ数が一切計測されていません |

### 1-2. サイトの技術的な問題

| 問題 | 詳細 |
|------|------|
| **titleタグのバグ** | HTMLの記述ミスにより、ページタイトルが正しく表示されていない可能性があります。検索結果での表示に影響します |
| **住所が2種類混在** | 店舗案内では久留米市、構造化データ・フッターでは城南区の住所が表示されています。Googleの評価に悪影響があります |
| **電話番号が2種類混在** | PCでは090-8451-0837、スマートフォンでは080-3935-2300が表示されています |
| **PCの電話番号がタップできない** | PC・タブレットで電話番号をクリックしても発信できません |

### 1-3. お問い合わせに繋がりにくい

| 問題 | 詳細 |
|------|------|
| **スマートフォンに固定の電話ボタンがない** | スクロールすると電話番号が画面から消えます。蜂駆除は「今すぐ電話したい」方が多いため、常に電話ボタンが見える状態が必要です |
| **お問い合わせへの導線が弱い** | 電話番号やお問い合わせフォームへの誘導が少なく、ページを見ているうちに離脱してしまいます |
| **施工事例・お客様の声がない** | 「ここに頼んで大丈夫かな？」という不安を解消するコンテンツがありません |

### 1-4. コンテンツの不足

| 問題 | 詳細 |
|------|------|
| **ページ数が少ない** | 公開中の固定ページは9ページのみ。競合は数十〜数百ページを展開しています |
| **地域別ページがない** | 「福岡市 蜂駆除」「久留米 蜂駆除」などの地域名での検索に対応できていません |
| **蜂の種類別ページがない** | 「スズメバチ 駆除」「アシナガバチ 駆除」などの検索に対応できていません |
| **ブログ記事のSEO設定が未設定** | 28件のブログ記事がありますが、全てSEOのtitle・descriptionが空です |
| **非公開のページが3ページある** | 「料金」「作業の流れ」「事例紹介」のページが非公開のまま眠っています |
| **鍵交換サービスの存在感が薄い** | ナビゲーションにリンクはありますが、トップページでの訴求が弱いです |

## 2. デモサイトでの改善点

### 2-1. 検索で見つかるための改善

| 改善内容 | 効果 |
|---------|------|
| **全ページのtitle・meta descriptionを最適化** | 検索結果に正しく表示され、クリックされやすくなります |
| **地域別ページ12ページを新規作成** | 「福岡市 蜂駆除」「久留米 蜂駆除」「北九州 蜂駆除」等の検索で上位表示を狙えます |
| **蜂の種類別ページ3ページを新規作成** | 「スズメバチ 駆除」「アシナガバチ 駆除」等の検索からの流入を獲得します |
| **titleタグのバグを修正** | Googleがページを正しく認識できるようになります |
| **住所・電話番号の統一** | NAP（名前・住所・電話番号）の一致で、Googleマップ検索での評価が改善します |

### 2-2. お問い合わせに繋がるための改善

| 改善内容 | 効果 |
|---------|------|
| **スマートフォンの画面下部に電話ボタンを常時表示** | どのページを見ていても、いつでもワンタップで電話できます |
| **全ページに電話番号とお問い合わせリンクを配置** | ページの途中・末尾にも問い合わせ導線を設置し、離脱を防ぎます |
| **料金表を見やすく整理** | 蜂の種類別・季節別の料金を一覧で確認でき、「いくらかかるか」の不安を解消します |
| **調査費用の注記を明記** | 「現地調査のみは1万円、駆除を行う場合は割引」という仕組みを明示し、安心感を高めます |
| **作業の流れを4ステップで説明** | 「お電話→現地調査→駆除→完了」の流れが一目でわかります |

### 2-3. 信頼感を高めるための改善

| 改善内容 | 効果 |
|---------|------|
| **施工事例ページの新設** | 実際の駆除写真と料金を掲載し、「ここなら安心」と感じていただけます |
| **よくある質問を3問→14問以上に拡充** | 料金・対応・作業・鍵交換など、お客様が気になる点を網羅します |
| **代表挨拶セクションの充実** | 「どんな人が来るのか」がわかり、安心して依頼できます |
| **鍵交換サービスもトップページで紹介** | 蜂駆除だけでなく、暮らしのトラブルを幅広く対応していることが伝わります |

### 2-4. サイト全体の改善

| 改善内容 | 効果 |
|---------|------|
| **デザインをシンプル・清潔に統一** | 信頼感のある見た目に。派手な広告感をなくし、誠実さが伝わるデザインです |
| **スマートフォン対応の強化** | スマートフォンからの閲覧が7〜8割を占めるため、スマートフォンでの見やすさ・使いやすさを優先しています |
| **共通パーツの一元管理** | ヘッダー・フッター等は1ファイルで管理するため、修正漏れや不整合が起きません |
