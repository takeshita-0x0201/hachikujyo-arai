## 1. 現在のサイトの課題

### 1-1. 検索で見つからない

| 課題 | 詳細 |
|------|------|
| Googleに認識されていない | サイト内に39ページあるのに、Googleが認識しているのは**3ページだけ**です |
| 「福岡 蜂駆除」で検索しても出てこない | 検索上位は全国チェーンやポータルサイトが占めており、蜂駆除あらいは圏外です |
| アクセス解析が入っていない | サイトへの訪問者数、電話タップ数、お問い合わせ数が**一切わからない**状態です |
| SEOプラグインが機能していない | All in One SEOが入っていますが、ブログ28件のSEO設定が**全て空**です |
| 地域名での検索に対応できていない | 「福岡市 蜂駆除」「久留米 蜂駆除」等のページが存在しません |

### 1-2. サイトの技術的な問題

| 課題 | 詳細 |
|------|------|
| **titleタグのバグ** | HTMLの記述ミスにより、ページタイトルが正しく出力されていません。検索順位に直結する致命的な問題です |
| **住所が2種類混在** | 店舗案内では久留米市、フッターやGoogleへの送信データでは城南区の住所になっています |
| **電話番号が2種類混在** | PCでは090-8451-0837、スマートフォンでは080-3935-2300が表示されています |
| **PCで電話番号がタップできない** | PC・タブレットで電話番号をクリックしても発信できません |
| **せっかく作ったページが非公開** | 「料金」「作業の流れ」「事例紹介」の3ページが非公開のまま眠っています |

### 1-3. お問い合わせに繋がりにくい

| 課題 | 詳細 |
|------|------|
| スマートフォンに固定の電話ボタンがない | スクロールすると電話番号が画面から消えます |
| 施工事例・お客様の声がない | 「ここに頼んで大丈夫か」の判断材料がありません |
| 蜂駆除は緊急サービスなのに、緊急性が伝わらない | 「今すぐ電話したい」人への導線が弱いです |

## 2. リニューアルでの改善点

### 2-1. 検索で見つかるための改善

| 改善 | 効果 |
|------|------|
| **全ページのtitle・meta descriptionを最適化** | 「福岡 蜂駆除」等のキーワードで検索結果に正しく表示されます |
| **地域別ページ17ページを新規作成** | 福岡市7区・久留米市・北九州市・春日市・大野城市・筑紫野市・太宰府市・飯塚市・佐賀県・熊本県、それぞれの地域名で検索上位を狙えます |
| **蜂の種類別ページ3ページを新規作成** | 「スズメバチ 駆除」「アシナガバチ 駆除」等の検索からの流入を獲得します |
| **ページ間の内部リンクを整備** | 蜂駆除ページ→種類別ページ→地域ページが相互に繋がり、Googleの評価が上がります |
| **構造化データを全ページに追加** | Googleに「蜂駆除の事業者」であることを正確に伝えます |
| **titleタグのバグを修正** | Googleがページを正しく認識できるようになります |
| **住所・電話番号を統一** | Googleマップ検索での評価が改善されます |

### 2-2. お問い合わせを増やすための改善

| 改善 | 効果 |
|------|------|
| **スマートフォンの画面下部に電話ボタン+電話番号を常時表示** | どのページでも、いつでもワンタップで電話できます |
| **全ページに電話番号とお問い合わせリンクを複数配置** | 料金表の下、ページの末尾など、複数箇所に配置して離脱を防ぎます |
| **料金表を見やすく整理** | 蜂の種類別・季節別の料金を一覧で確認でき、電話前の不安を解消します |
| **調査費用の仕組みを明記** | 「現地調査のみは1万円、駆除を行う場合は割引」を料金表の直下に表示します |
| **よくある質問を3問→15問以上に拡充** | 料金・対応・作業・鍵交換について、気になる点を網羅します |
| **施工事例ページを新設** | 実際の駆除写真と料金を掲載し、安心感を高めます |

## 3. 追加施策のご提案

サイトリニューアルに加えて、以下の施策を実施することで問い合わせ数の増加が見込めます。

### 3-1. Google Analytics / Search Console の導入

| ツール | 効果 | 費用 |
|--------|------|------|
| **Google Analytics 4** | サイトの訪問者数、どのページが見られているか、電話ボタンのタップ数がわかるようになります | 無料 |
| **Google Tag Manager** | 電話タップ・フォーム送信を「問い合わせ」として計測できます | 無料 |
| **Google Search Console** | どんなキーワードで検索されているか、検索結果で何回表示されているかがわかります | 無料 |

**現状、これらが一切入っていないため、サイトの効果がまったく測れません。** リニューアルと同時に導入します。

### 3-2. Googleビジネスプロフィール（Googleマップ）の最適化

「福岡 蜂駆除」で検索すると、検索結果の上部にGoogleマップが表示されます。ここに表示されるかどうかで、電話の数が大きく変わります。

| 施策 | 効果 |
|------|------|
| **ビジネス情報の完全記入** | 住所・電話番号・営業時間・サービス内容を正確に登録します |
| **施工写真のアップロード（50枚以上）** | 写真が多い事業者は検索表示回数が大幅に増えます |
| **口コミの獲得** | 口コミ数が増えるとGoogleマップ検索での順位が上がります |
| **定期的な投稿** | 週1回、施工事例や季節の情報を投稿します |

**特に重要なのが口コミです。** 施工完了後にお客様に口コミをお願いする仕組みを作ります。

| 口コミ獲得の方法 | 内容 |
|----------------|------|
| **口コミ依頼カード** | 施工後にお渡しするカード。QRコードでGoogleレビューページに直接アクセスできます |
| **SMS/LINE依頼** | 施工後にテキストメッセージで口コミ依頼を送ります |
| **口コミへの返信** | いただいた口コミには全て返信します。返信がある事業者は信頼度が上がります |

### 3-3. LINE公式アカウントの導入

電話が苦手な方や、営業時間外に相談したい方のために、LINEでのお問い合わせ窓口を設けることをご提案します。

| メリット | 詳細 |
|---------|------|
| 電話が苦手な層をカバー | 特に若い世代は電話よりLINEを好む傾向があります |
| 写真で相談できる | 蜂の巣の写真を送ってもらえば、種類や概算料金をお伝えできます |
| 営業時間外の受付 | 深夜や早朝の相談も取りこぼしません |
| リピーター獲得 | LINE登録者に季節の注意喚起を配信できます |

| 費用 | 内容 |
|------|------|
| LINE公式アカウント | **無料**（フリープラン：月200通まで） |
| サイトへの導入 | スマートフォンの固定ボタンにLINEアイコンを追加します |

### 3-4. ポータルサイトへの掲載

自社サイトのSEOが強くなるまでの間、ポータルサイト経由でお問い合わせを獲得する方法です。

| サイト | 手数料 | メリット |
|--------|--------|---------|
| **くらしのマーケット** | 予約手数料20%（税込22%） | 口コミが蓄積される。登録翌日から問い合わせ獲得可能 |
| **ミツモア** | 成約課金22%（税込） | 自動応募機能あり。リピート客獲得率60% |

手数料はかかりますが、**自社サイトのSEOが育つまでの即効性のある集客チャネル**として有効です。

### 3-5. サーバー移行

現在、他の業者様がサーバーを管理されているとのことですので、自社契約のCoreserver CORE-Yへの移行をご提案します。

| メリット | 詳細 |
|---------|------|
| **コスト削減** | 他社への管理費が不要になります |
| **自由に更新できる** | ページの追加・修正を自分たちで行えます |
| **ドメインはそのまま** | hachikujyo-arai.com のアドレスは変わりません。SEO評価も引き継がれます |
